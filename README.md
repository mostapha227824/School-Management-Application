 School Management App

Overview
The **School Management App** is a mobile application designed to streamline student, teacher, and administrative interactions. It provides an efficient way to manage attendance, grades, fees, and leave requests while ensuring a seamless experience for all users.

Features
-  **User Authentication**: Secure login and registration system.
-  **Student Dashboard**: View attendance, grades, transcripts, and fee details.
-  **Teacher Dashboard**: Mark and view attendance, manage grades.
-  **Fee Payment & Billing**: Integrated online payment for school fees.
-  **Leave Requests**: Students can submit leave requests with optional supporting documents.
- âš™ **Admin Controls**: Manage attendance records, fees, and users.

 Technologies Used
- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Firestore & Authentication)
- **Database**: Firebase Firestore
- **Authentication**: Firebase Authentication
- **Cloud Storage**: Firebase Storage (for documents)

## Installation & Setup
### **Prerequisites**
Ensure you have the following installed:
- [Flutter](https://flutter.dev/docs/get-started/install)
- [Dart](https://dart.dev/get-dart)
- Firebase project setup ([Guide](https://firebase.google.com/docs/flutter/setup))
- Android Studio or VS Code with Flutter extensions

 **Clone the Repository**
```sh
git clone https://github.com/yourusername/school-management-application.git
cd school-management-application
```

 **Install Dependencies**
```sh
flutter pub get
```

 **Run the App**
```sh
flutter run
```
